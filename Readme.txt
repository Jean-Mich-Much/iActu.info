iactu.info — Version 18
Date : 26 mars 2026

Code source de la version 18 du site iactu.info.

Technologies utilisées :
- PHP 8.3.6
- HTML5
- CSS3
- Python
- Ubuntu 24.04 LTS
- Serveur Caddy

Documentation :
Une documentation complète est disponible dans le dossier :
Fondation/doc/

Elle décrit l’architecture, les scripts CLI, la gestion des flux RSS, les conventions de nommage, les procédures de mise à jour et les prérequis système.

Structure du projet :
- index.php : point d’entrée
- jeux.php : logique interne et modules
- maj_cli.php : mises à jour automatisées
- fusion_rss.php : fusion et normalisation des flux RSS
- assets/ : ressources statiques
- Fondation/ : cœur technique et documentation

Environnement recommandé :
- Linux (Ubuntu 24.04)
- PHP 8.3.x
- Serveur Caddy
- Cron ou systemd timers

Notes :
- Le projet respecte une logique stricte
- Les scripts CLI sont conçus pour fonctionner en environnement Linux uniquement  
- Le code est organisé pour être lisible, stable et facilement maintenable
